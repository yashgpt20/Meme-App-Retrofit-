# Meme App 😂

Heyy Folks! 👋 Welcome to my Meme App! 

## 🚀 About the Project
This is a basic Android application built to fetch and display random memes from an external API. The primary purpose behind building this project was to get hands-on experience with API calling and to become familiar with integrating external network data into an Android app.

## ✨ How it Works
* The app connects to an external meme API to retrieve data.
* Simply click the **"New Meme"** button, and the app will instantly make a network request to load and display a brand new meme on your screen!

## 🛠️ Tech Stack
* **Environment:** Android Studio
* **Networking:** Retrofit (Used for seamless API integration and JSON parsing)
* **Language:** Kotlin
* **UI:** XML Layouts
